# Workshop.js

JavaScript workshop based on CLI inquiry

## Getting Started
Install the module with: `npm install workshopjs`

Launch the workshop with:
```javascript
var workshop = require('workshop');
workshop({
 cwd: "your_workshop_dir"
 validators: {...your question resolvers...}
});
```
